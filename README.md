# 🧠 Guardian – A Caring Companion for Dyslexia

An innovative web-based platform designed to support dyslexic children by providing personalized and engaging learning experiences to improve reading and comprehension skills.

---

## 📌 Project Overview

Many children with dyslexia face challenges in reading, comprehension, and academic performance, which can impact their confidence and learning journey.

**Guardian** is developed to address these challenges by offering a supportive and interactive environment that adapts to each child’s learning needs.

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL (Database)

---

## ✨ Features

### 📖 Engaging Storytelling

* Interactive and easy-to-read stories designed for dyslexic learners
* Improves reading skills in a fun and engaging way

---

### 🧩 Interactive Quizzes

* Quizzes to evaluate reading and comprehension levels
* Classifies performance into:

  * Low
  * Average
  * High

---

### 🎯 Personalized Learning Paths

* Adaptive learning based on user performance
* Customized content to improve weak areas

---

### 📊 Parent Analytics Dashboard

* Track child’s progress and improvement
* Monitor reading performance over time

---

## 🧠 Key Benefits

* Enhances reading ability and comprehension
* Builds confidence in learners
* Provides structured and personalized learning
* Helps parents stay informed about progress

---

## 🎯 Purpose

This project aims to:

* Support dyslexic children with tailored learning tools
* Improve reading and comprehension skills
* Provide an interactive and engaging educational experience
* Enable parents to track and support their child’s learning journey
